Detailed Description

Continuous acquisition configuration. Provided to ads1299_start_continuous(). All buffer sizing is derived automatically from config.sample_rate and chunk_duration_ms — the caller does not manage any buffers directly.

◆ ring_buffer_chunks

uint32_t ads1299_continuous_config_t::ring_buffer_chunks

Ring buffer depth in number of chunk slots. 0 = ADS1299_RING_BUF_SLOTS (default).

Must be a power of 2. Rounded up internally if not.

Total memory allocated: ring_buffer_chunks * chunk_samples * sizeof(ads1299_sample_t)

At 250 SPS, 100 ms chunks, 8 slots: 8 * 25 * 44 = 8,800 bytes

At 16 kSPS, 10 ms chunks, 8 slots: 8 * 160 * 44 = 56,320 bytes

At 16 kSPS, 100 ms chunks, 8 slots: 8 * 1600 * 44 = 563,200 bytes ← EXCEEDS ESP32 SRAM; reduce chunk_duration_ms
