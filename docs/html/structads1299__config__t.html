Detailed Description

Driver configuration structure.

Static device configuration. Provided once at ads1299_create(). The SPI bus (spi_bus_initialize) must be initialised by the caller before ads1299_init() is called. The driver only calls spi_bus_add_device() / spi_bus_remove_device() internally.
